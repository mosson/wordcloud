#### to Boot

別途 config/application.yml を設置して以下を記入する必要がある

to boot, you have to set config/application.yml


```

SECRET_TOKEN: APPLICATION_SECRET_TOKEN

TWITTER_ACCESS_TOKEN: 
TWITTER_SECRET_KEY: 
FACEBOOK_ACCESS_TOKEN: 
FACEBOOK_SECRET_KEY: 

AWS_ACCESS_TOKEN: 
AWS_SECRET_KEY: 
AWS_ACCESS_REGION: 
AWS_BUCKET_NAME: 

```
